# PythonFixsender
FIX Sender in Python using simplefix &amp; tkinter

It's challenging to find fully functional working FIX Sender using Python so I build my own using AI. 

Setup and startup Instruction
1) pip in simplefix
2) Python main.py
3) In Connection tab, Start the FIX Server

<img width="397" height="760" alt="image" src="https://github.com/user-attachments/assets/465fa1d3-b5f2-494e-8539-326d76788999" />

4) In the Connection tab, Connect the FIX Client

<img width="332" height="414" alt="image" src="https://github.com/user-attachments/assets/d3f0dd98-a5f5-4828-b43d-89df22162efc" />

If you are using different setting, pls make sure to update the SenderCompID/TargetCompID/port in both the FIX Client and FIX Server 

5) To send order, go to the <Order Entry> Tab
   
<img width="370" height="341" alt="image" src="https://github.com/user-attachments/assets/6027d6b5-0a8c-4882-8a0d-1a3362f4b8d5" />

Send Order

6)  Go to the <Client Order> tab to see client order

<img width="1196" height="218" alt="image" src="https://github.com/user-attachments/assets/729dab2f-bdbd-4014-82f5-65255634e0e3" />


7) Order is in Pending status which need to be manually acknowlodged in the <Server Order> tab

<img width="1191" height="776" alt="image" src="https://github.com/user-attachments/assets/431416a7-20e0-483c-b720-a3d5dea0f176" />

Pls accepted before executed the order or alternatively reject the order.
